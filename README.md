## 📝 Microsoft Word

**Microsoft Word** es el procesador de textos más utilizado en el mundo, parte de la suite de Microsoft 365.

### ¿Qué aprendí?

- Crear y dar formato a documentos profesionales
- Usar estilos y plantillas prediseñadas
- Insertar tablas, imágenes y gráficos
- Trabajar con encabezados, pies de página y numeración
- Usar el corrector ortográfico y gramatical
- Exportar documentos en diferentes formatos (PDF, DOCX, TXT)
- Colaborar en tiempo real con otros usuarios

### Herramientas más útiles

- ✅ Control de cambios para revisiones
- ✅ Tabla de contenidos automática
- ✅ Combinación de correspondencia
- ✅ Macros para automatizar tareas repetitivas

## 📄 Google Docs

**Google Docs** es un procesador de textos en línea y gratuito que forma parte de Google Workspace.

### ¿Qué aprendí?

- Crear documentos directamente desde el navegador sin necesidad de instalar software
- Compartir documentos con permisos personalizados (ver, comentar, editar)
- Colaborar en tiempo real con múltiples usuarios simultáneamente
- Ver el historial de versiones y restaurar versiones anteriores
- Usar complementos para ampliar las funcionalidades
- Sincronización automática en Google Drive

### Comparación con Word

| Característica | Google Docs | Microsoft Word |
|---|---|---|
| Costo | Gratuito | De pago |
| Colaboración en tiempo real | ✅ Excelente | ✅ Disponible |
| Acceso sin instalación | ✅ Sí | ❌ No |
| Funciones avanzadas | Básicas | Avanzadas |
| Almacenamiento | Google Drive | OneDrive |

## 📑 LovePDF

**LovePDF** es una plataforma en línea especializada en la gestión y edición de archivos PDF de manera sencilla y gratuita.

### ¿Qué aprendí?

- Unir varios archivos PDF en uno solo
- Dividir un PDF en páginas o secciones independientes
- Comprimir PDFs para reducir su tamaño sin perder calidad
- Convertir PDFs a Word, Excel, PowerPoint y viceversa
- Rotar, reorganizar y eliminar páginas de un PDF
- Proteger documentos PDF con contraseña
- Firmar digitalmente documentos

### Herramientas disponibles

| Herramienta | Función |
|---|---|
| Merge PDF | Une múltiples PDFs en uno |
| Split PDF | Divide un PDF por páginas |
| Compress PDF | Reduce el tamaño del archivo |
| PDF to Word | Convierte PDF a documento editable |
| Word to PDF | Convierte documentos Word a PDF |
| Protect PDF | Añade contraseña al documento |

## 🎯 Conclusión

El dominio de estas herramientas digitales representa una ventaja significativa en el entorno académico y profesional actual. Cada una cumple un propósito específico:

- **Microsoft Word** → Para documentos profesionales y formateados

- **LovePDF** → Para gestión y edición de archivos PDF

> 🚀 El aprendizaje continuo de estas herramientas nos permite ser más eficientes, creativos y productivos en nuestras actividades diarias.
